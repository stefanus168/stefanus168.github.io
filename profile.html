Portofolio

Forum Gaming and Chat (FNC)

FNC merupakan sebuah platform komunitas yang disediakan untuk para pemain games online seperti valorant,mobile legends, dan dota 2. Website ini disediakan secara gratis dan hanya berlaku sampai tahun depan sekitar bulan april.

Bagi yang ingin mencoba dapat Klik link ini

Sign In

Sign In merupakan salah satu fitur yang saya buat, dimana user dapat melakukan login menggunakan username dan password yang didaftarkan atau menggunakan gmail

Sign Up

Sign Up merupakan salah satu fitur yang saya buat, dimana user dapat melakukan register dengan melengkapi form dan datanya akan disimpan di database MySQL

Homepage

Pada bagian Homepage saya membuat 3 fitur diantaranya adalah rekomendasi friend merupakan fitur yang dibuat supaya user dapat menemukan teman yang memiliki rating bagus,latest forum merupakan fitur untuk melihat informasi forum yang terakhir dibuat user,news merupakan fitur yang ditambahkan oleh admin agar user mendapatkan update mengenai game.

Forum

Pada bagian forum saya membuat 2 fitur yaitu create dan view list,User dapat membuat forum atau melihat list forum sesuai jenis game yang dipilih nanti akan ditampilkan forum-forum yang telah dibuat oleh user lain.

Detail Forum

Pada bagian detail forum saya membuat view detail dan juga reply, berisikan deskripsi forum yang telah dibuat oleh user lain dan user lain dapat berkomentar juga pada postingan tersebut dengan menekan reply.

Build

Build merupakan salah satu fitur yang saya buat,User dapat melihat build sesuai game yang dipilih dan nanti akan menampilkan list hero-hero dan jika di klik akan menampilkan hero dengan item yang cocok, terdapat 3 jenis game ml,valorant,dan dota 2.

Group Forum

Pada Group forum saya membuat 3 sub menu, yang pertama untuk membuat group, yang kedua untuk menampilkan list group selain itu user juga dapat melakukan update atau delete pada group yang sudah dia buat, yang ketiga ketika user mendapat invite join group dapat menerima atau menolak.

Detail Group

Pada detail group saya membuat view description, replygroup, dan invite friend to group. Reply group dibuat agar yang tergabung pada group dapat berkomentar di group tersebut, sedangkan invite friend to group hanya bisa jika user sudah melakukan pertemanan dengan orang yang ingin diundang

Personal Chat

Pada personal chat saya membuat 3 fitur yaitu add friend yang digunakan untuk mengundang orang pada list yang ditampilkan untuk menjadi teman atau dapat mencari by id, Yang kedua merupakan personal chat yang digunakan agar user dapat bertukar pesan dengan orang yang sudah menjadi temannya, yang ketiga merupakan list acceptance dimana user dapat menerima atau menolak permintaan teman
